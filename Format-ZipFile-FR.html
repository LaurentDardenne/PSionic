Name

Format-ZipFile

SYNOPSIS

Formate le contenu des propriétés d'un objet archive.

SYNTAX

Format-ZipFile -Zip <ZipFile> [[-Properties] <Object>] [<CommonParameters>]

DESCRIPTION

Formate le contenu des propriétés d'une archive Zip.

Consultez la rubrique Notes pour le détail des propriétés affichées par défaut.

PARAMETERS

-Zip <PSObject>

Objet archive sur lequel appliquer le formatage.

Vous devez forcer la prise en compte de l'objet en préfixant la variable par une virgule.

EXAMPLES

-------------- EXAMPLE 1 --------------

PS C:\> try {         
,($ZipFile=Get-Zipfile -Path Test.zip)|Format-ZipFile
} finally {
$ZipFile.PSDispose()
}

Cet exemple récupère un objet archive à partir d'un nom de fichier, puis formate son contenu avant de libérer les ressources systèmes.
