Format-ZipFile

Synopsis

Formats content of the properties of an archive object.

Syntax

Format-ZipFile -Zip <PSObject> [[-Properties] <Object>] [<CommonParameters>]

Description

Formats content of the properties of a Zip archive. Read Notes section for details about default displayed properties.

Parameters

-Zip <PSObject>

Archive object to format. You need to force the inclusion of the object by prefixing the variable with a comma.

Required? True
Position? named
Default value  
Accept pipeline input? true (ByValue)
Accept wildcard characters? False

-Properties <Object>

Array of property names to display when internal calling to Format-List.

Required? False
Position? 0
Default value  
Accept pipeline input? false
Accept wildcard characters? False

<CommonParameters>

This cmdlet supports the common parameters: Verbose, Debug,
ErrorAction, ErrorVariable, WarningAction, WarningVariable,
OutBuffer, PipelineVariable, and OutVariable. For more information, see
about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

Examples

EXAMPLE 1

PS C:\> try {         
,($ZipFile=Get-Zipfile -Path Test.zip)|Format-ZipFile
} finally {
$ZipFile.PSDispose()
}

This example gets an archive object from a file name, then formats his contents before freeing system resources.
