ConvertFrom-CliXml

Synopsis

Creates a Powershell object from an XML representation and issues it in the pipeline.

Syntax

ConvertFrom-CliXml [-InputObject] <String[]> [<CommonParameters>]

Description

You can receive an XML object from the &;39;Expand-ZipEntry&;39; function.

Parameters

-InputObject <String[]>

Specifies XML representation of the object to be converted.

Required? True
Position? 0
Default value  
Accept pipeline input? true (ByValue)
Accept wildcard characters? False
