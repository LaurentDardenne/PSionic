Set-PsIonicSfxOptions

Synopsis

Assigns new values ​​to the default configuration.

Syntax

Set-PsIonicSfxOptions [-Options] <SelfExtractorSaveOptions> [<CommonParameters>]

Description

Assigns new values ​​to the default configuration by copying the values ​​of the configuration shown in the default configuration.

Parameters

-Options <SelfExtractorSaveOptions>

Contains the new values ​​to be assigned to the default configuration.

Required? True
Position? 0
Default value  
Accept pipeline input? true (ByValue)
Accept wildcard characters? False

<CommonParameters>

This cmdlet supports the common parameters: Verbose, Debug,
ErrorAction, ErrorVariable, WarningAction, WarningVariable,
OutBuffer, PipelineVariable, and OutVariable. For more information, see
about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

Inputs

Ionic.Zip.SelfExtractorSaveOptions

Outputs

None

Notes

Examples

EXAMPLE 1

PS C:\> $MyConfiguration=New-ZipSfxOptions -Copyright "This module is free for non-commercial purposes." 
Set-PsIonicSfxOptions $MyConfiguration

Those instructions create a SFX configuration with default values except for the Copyright field. Then it assigns new values ​​to the default configuration.
