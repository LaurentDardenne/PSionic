Reset-PsIonicSfxOptions

Synopsis

Resets the default configuration used in the construction of a self-extracting archive.

Syntax

Reset-PsIonicSfxOptions [<CommonParameters>]

Description

Resets the default configuration used in the construction of a self-extracting archive. This default configuration is used internaly from public functions of the Psionic module, related to the management of self-extracting archive.

Parameters

<CommonParameters>

This cmdlet supports the common parameters: Verbose, Debug,
ErrorAction, ErrorVariable, WarningAction, WarningVariable,
OutBuffer, PipelineVariable, and OutVariable. For more information, see
about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

Inputs

None

Outputs

None

Notes

Examples

EXAMPLE 1

PS C:\> Reset-PsIonicSfxOptions

This instruction resets the default configuration.
