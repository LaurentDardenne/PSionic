Get-PsIonicSfxOptions

Synopsis

Returns a default configuration copy used when building a self-extracting archive.

Syntax

Get-PsIonicSfxOptions [<CommonParameters>]

Description

Returns a default configuration copy used when building a self-extracting archive. This default options set is used internally from differents public functions of Psionic module.

Parameters

<CommonParameters>

This cmdlet supports the common parameters: Verbose, Debug,
ErrorAction, ErrorVariable, WarningAction, WarningVariable,
OutBuffer, PipelineVariable, and OutVariable. For more information, see
about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

Inputs

None

Outputs

Ionic.Zip.SelfExtractorSaveOptions

Notes

Examples

EXAMPLE 1

PS C:\> $MyConfiguration=Get-PsIonicSfxOptions

This instruction returns the default configuration currently set.
