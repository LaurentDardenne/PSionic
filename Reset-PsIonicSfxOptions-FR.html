Name

Reset-PsIonicSfxOptions

SYNOPSIS

Réinitialise la configuration par défaut utilisée lors de la construction d'une archive auto extractible.

SYNTAX

Reset-PsIonicSfxOptions [<CommonParameters>]

DESCRIPTION

Réinitialise la configuration par défaut utilisée lors de la construction d'une archive auto extractible.

Cette configuration par défaut est utilisée en interne par les différentes fonctions publiques du module PsIonic, liées à la gestion d'archive auto extractible.

<CommonParameters>
